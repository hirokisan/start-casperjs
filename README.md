# Start casperjs

## install
```
brew install casperjs
```

## Run php server
```
php -S localhost:8080
```

## Run casperjs
```
casperjs test.js
```

## Notice
- [cant use coffeeScript above phantomjs2.0](http://docs.casperjs.org/en/latest/known_issues.html#known-issues)
